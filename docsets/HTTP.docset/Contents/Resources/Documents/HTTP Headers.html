HTTP Headers

  • HTTPHeaders

    Representation of the HTTP headers associated with a HTTPRequest or HTTPResponse. Headers are subscriptable using case-insensitive comparison or provide Name constants. eg. swift let contentLength = headers["content-length"] or swift let contentLength = headers[.contentLength]
